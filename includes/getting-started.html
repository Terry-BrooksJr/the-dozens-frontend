Getting Started With The Yo' Momma Jokes API
Developer-friendly by design. The Yo' Momma Jokes API offers clean, well-documented endpoints with clear error messages, generous rate limiting, and dual REST + GraphQL interfaces — so you can get laughing in minutes.

What You Can Do

Random Jokes

Fetch a random Yo' Momma joke on demand — no auth required for read access.

Filter by Category & NSFW

Narrow results by joke category and explicit content flag to match your audience.

REST & GraphQL

Choose between a traditional REST API or the modern GraphQL interface — same data, your preference.

Contribute Jokes

Create an account to submit new jokes and help grow the collection.

Create an Account Optional

Required only if you plan to contribute jokes. Just reading? Skip to Step 3.

Register a New User
POST /auth/users/
Request Body
{
  "email": "user@example.com",
  "username": "dozens_lover",
  "password": "securePassword123"
}
Response 201 Created 
{
  "id": 12,
  "username": "dozens_lover",
  "email": "user@example.com"
}
Obtain an Auth Token

Use your credentials to get a token for write operations:

POST /auth/token/login/
Request Body
{
  "username": "dozens_lover",
  "password": "securePassword123"
}
Response 200 OK 
{
  "auth_token": "9944b09199c62bcf..."
}
Include your token in the Authorization header as Token 9944b09199c62bcf... for all write requests.

Fetch Jokes

No authentication required for read endpoints.

Get a Random Joke
GET /api/insults/random/
Optional Query Parameters
Parameter Type Example Description
category string P Filter by joke category (e.g. P = Poor)
nsfw boolean false Include explicit content (true) or keep it clean (false)
Example Request
GET /api/insults/random/?category=P&nsfw=false
Response 200 OK 
{
  "reference_id": "GIGGLE_ABC123",
  "content": "Yo mama is so poor, she
    can't even afford to pay
    attention.",
  "category": "Poor",
  "nsfw": false,
  "added_by": "John D.",
  "added_on": "2 days ago"
}
Headers
Content-Type: application/json
cURL Example
curl -X GET \

application/json"
List All Jokes
GET /api/insults/

Returns a paginated list of all jokes. Supports the same category and nsfw query parameters.
